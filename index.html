Einstieg in das Programmieren mit Python

In diesem Lernkursabschnitt beginnen wir mit dem tatsächlichen Programmieren. Dafür müsst ihr zuerst in der Python IDLE, oder eurer alternativen Entwicklungsumgebung, ein neues Dokument öffnen. Bei der Python IDLE könnt ihr das entweder mit CTRL+N oder File->"New File".


Beim Programmieren genügt es nicht reinzuschreiben "sage mir was das Resultat von 5+10 ist", denn das versteht Python nicht. Python funktioniert nur mit bestimmten "Befehlen". Der entsprechende Befehl dafür wäre: print(5+10), aber dazu kommen wir später.

Semantik und Syntax

Zwei wichtige Begriffe sind die Syntax und die Semantik. Die Syntax beschreibt wie bestimmte Zeichenfolgen (bzw. Wörter und/oder Sätze) formuliert sein müsse. Ein Beispiel für eine Syntax ist die Grammatik der deutschen Sprache. Nach der Syntax der deutschen Sprache (=Grammatik) ist zum Beispiel "Haste mer nen Schokoriegel" nicht korrekt, gleich wie print 5+10 nach der Syntax von Python ebenfalls nicht korrekt ist, da die Klammern vor 5+10 fehlen. Die Semantik hingegen ist die Bedeutung der Zeichen, Wörter und so weiter. So bedeutet zum Beispiel print(5+10), dass Python das Ergebnis von 5+10 in der Shell ausgegeben werden sollte. Wenn wir dieses Beispiel wieder auf die deutsche Sprache, bedeutet "Könnte ich einen Schokoladenriegel haben?", dass die Person einen Schokoladenriegel haben möchte und man ihm doch bitte diesen Schokoladenriegel geben sollte.

Kommentare

Etwas sehr Wichtiges, welches beim Programmieren gerne mal vergessen wird ist das Kommentieren. Kommentare können, besonders bei großen Programmen, uns helfen den Überblick zu bewahren. Außerdem kann man dank diesen Kommentaren oft nachzuvollziehen was man sich bei diesem Schritt überlegt hat. Wenn man vor dem Kommentar ein "#" einfügen, der Rest der Zeile wird bei der Ausführung ignoriert.

Datentypen & Variabeln

An dieser Stelle würde ich gerne einen Exkurs über die verschiedenen Datentypen einfügen. Wie ihr sicherlich schon bemerkt habt habe ich Hello World mit Anführungs- und Schlusszeichen bei der Erklärung gemacht, bei der allgemeinen Erklärung fehlen diese jedoch. Denn die Anführungs- und Schlusszeichen gehören nicht etwa zum Befehl, sondern werden dafür gebraucht um dem Computer mitzuteilen, dass "Hello World" ein Wort und nicht etwa eine Variable oder eine Zahl ist. Es gibt bei Python verschiedene Datentypen. Die wichtigsten Datentypen für den Anfang sind Strings, Integrer, Floats und Listen. Strings sind Wörter, beziehungsweise Zeichenketten. Generell sind das Wörter, aber auch Zahlen können Zeichenketten sein. Diese werden durch Anführungs- und Schlusszeichen gekennzeichnet. Um einen anderen Datentyp in ein String umzuwandeln, könnt ihr den Befehl str() verwenden. str(53) wäre so zum Beispiel "53". Wenn man zwei Integrer addiert, werden sie zusammengefügt. Zum Beispiel wird "3"+"3" zu "33". Integrer sind ganzzahlige Zahlen wie zum Beispiel 1, 3 oder 23. Um zum Beispiel eine Kommazahl auf die nächste ganzzahlige Zahl abzurunden und diese in einen Integrer umzuwandeln könnt ihr int() verwenden, int(2.99) ist zum Beispiel 2. Um (Gleit-)Kommazahlen werden gewöhnlicherweise als floats gespeichert. Um eine Zahl in eine Float umzuwandeln solltet ihr float() verwenden. Mithilfe von Listen kann man mehrere Werte in einer Liste speichern. Wie bei allen anderen Datentypen kann man diese danach verändern. Bei einer Liste können nach belieben auch verschiedene Datentypen gelöscht werden. Eine liste erstellt man mit eckigen Klammern. Ein Beispiel hierfür wäre die imaginäre Liste [1,"Apfel", 2.3]. Die verschiedenen Elemente der Liste kann man mit Liste[X]. X beginnt bei 0 und ist so gross wie es Elemente in der Liste hat (-1).

Datentypen

String --> Zeichenketten --> str(53) ist die Zeichenkette "53"


Integer --> ganzzahlige Zahlen --> Int(3) ist die ganzzahlige Zahl 3


Float --> Kommazahlen --> float(3.2) ist die Kommazahl 3.2


Liste --> mehre Werte in einer Liste speichern --> [1,"Apfel", 2.3] ist die Liste mit den Elementen 1, "Apfel" und 2.3

Variabeln sind Platzhalter für einen bestimmten Wert. Dieser kann ein String, Liste, eine Zahl oder einen anderen beliebigen Datentyp sein. Mithilfe von Variablen kann man einen Wert zwischenspeichern, welchen man später wieder aufrufen und verwenden kann. Ein Beispiel ist dafür die Variable X, wie Sie in der Mathematik verwendet wird. In Python ist der umgang mit Variablen signifikant einfacher als in anderen Sprachen. In anderen Sprachen muss man die Variable deklarieren. Dies bedeutet, dass man die Variable einem Typ zuordnet. In Python geschieht dies automatisch. Ausserdem kann in Python der Wert und der Datentyp der Variable im Laufe des Programms verändert werden. Doch wie weist man einer Variable einen Wert zu? Dies ist eigentlich ganz einfach, wenn man zum Beispiel x den Wert 18 zuordnen möchte, würde der Programmcode beispielsweise so aussehen:"x=18". Hier ist es allerdings wichtig zu merken, dass das "=" nicht das gleiche wie ein "=" in der Mathematik ist. Das "=" bedeutet hier lediglich, dass "X" den Wert 18 annimmt. Operatoren

Das grundlegenden Operatoren wie +,-,* und / (beziehungsweise "Plus","Minus","mal" und "geteilt durch") sind ziemlich selbsterklärend und dieselben wie auf einem gewöhnlichen Taschenrechner. Bei der Division sind die Datentypen wieder ein Thema. Normalerweise erhaltet man bei einer Division immer eine Float als Resultat. Wenn man allerdings ein Integrer als Resultat erhalten will muss man die sogenannte "flooring Division" verwenden. Das Operationszeichen für die "flooring Division" ist ein doppelter Bruchstrich (//).


Eher ungewöhnlich sind die Notationen für das potenzieren und den Modulo (Rest einer Divison). Denn diese sind Zahl**Exponent für die Potenz beziehungsweise Zähler%Nenner für den Modulo.

Operatoren

Summand+Summand -> addiert zwei Zahlen.


Minuend-Subtrahend -> subtrahiert zwei Zahlen.


Faktor*Faktor-> multipliziert zwei Zahlen.


Zähler/Nenner-> dividiert zwei Zahlen. Das Resultat ist eine Float (Kommazahl).


Zähler//Nenner->dividiert zwei Zahlen. Das Resultat ist ein Integrer (ganzzahlige Zahl).


Zahl**Exponent-> Zahl wird mit Exponent potenziert


Zähler%Nenner-> Modulo zweier Zahlen. Der Rest der Division ist das Resultat.

Begriffserklärungen

Software: Der Teil des Computers den man nicht anfassen kann. Software (syn. Programm) kann von einem Computer ausgeführt werden und sagt bestimmter Hardware (Sachen die man anfassen kann) was sie wie zu erledigen bzw. zu berechnen hat.

Binärcode/Maschinensprache: Maschinensprache besteht aus zwei "Zeichen" 1 und 0. Dies da der Computer nur 2 Zustände kennt: Strom fliesst (1), beziehungsweise kein Strom fliesst (0). Diese Maschinensprache kann dann von einem Computer ausgeführt werden.

Quellcode: Ein Text, welcher in einer Programmiersprache geschrieben worden ist und von Menschen gelesen werden kann. Der Computer kann die verschiedenen Befehle einer Programmiersprache in sogenannter Maschinensprache übersetzten, welche dann von einem Computer ausgeführt werden kann.
